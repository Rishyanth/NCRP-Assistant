# Steps to Follow

# Create a new enviornment
bash
`
conda create -n venv python==3.9.4
`
       
# Activate the virtual enviornment
bash
`
conda activate venv
`
# Install requirements
bash
`
pip install -r requirements.txt
`

# Run Streamlit application
bash
`
streamlit run app.py
`
